# NeuralGraph

Система верификации и распознавания рукописного текста.

### Особенности:
- Siamese Network: Определение авторства почерка (PyTorch).
- Line Segmentation: Геометрическая нарезка страниц на строки через OpenCV.

### Текущий прогресс:
- Обучение сиамской сети (точность верификации ~97%)
- Алгоритм сегментации строк (Contour-based)
- Интеграция с Telegram Bot (в процессе)
- Определение написанного при помощи OCR (в процессе)
